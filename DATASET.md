# Dataset

## Student Profile

The dataset for the learner profile contains the necessary information used by the system to personalize the learning experience and initialize the adaptive learning engine.

### Data Fields

- Learner ID
- Full Name
- Age
- Sex
- Grade Level
- Developmental Age
- Autism Level
- Existing Assessment Results
- Learning Preferences
- Sensory Preferences
- Teacher Assigned
- Parent/Guardian Information
- Previous Learning Progress
- Current Learning Level
- Reward Points
- Session History

---

## Lesson Materials / Content

The lesson dataset consists of structured developmental learning activities designed for learners with Autism Spectrum Disorder (ASD). Activities are categorized according to developmental domains and difficulty levels to support the adaptive learning engine.

### Learning Categories

#### Attention-Based Activities
- Object Identification
- Spot the Difference
- Visual Tracking
- Maze Activities
- Pattern Recognition

#### Emotion Recognition
- Identify Facial Expressions
- Match Emotions
- Emotion Scenarios
- Social Cue Recognition

#### Fine Motor Activities
- Letter Tracing
- Number Tracing
- Shape Tracing
- Drag-and-Drop Exercises
- Connect the Dots

#### Pre-Academic Skills
- Alphabet Recognition
- Number Recognition
- Color Identification
- Shape Identification
- Counting Exercises
- Matching Activities
- Sorting Activities

---

## Activity Metadata

Each learning activity should contain:

- Activity ID
- Activity Title
- Learning Category
- Difficulty Level
- Target Skill
- Instructions
- Learning Objectives
- Activity Content
- Correct Answers
- Feedback Messages
- Reward Points
- Estimated Completion Time
