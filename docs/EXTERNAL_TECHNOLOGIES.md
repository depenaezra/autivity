# Third-Party Technologies and External Components

This section identifies the external technologies, frameworks, libraries, and services utilized by AutiVity. These components are not developed by the researchers but are integrated to support system development, deployment, and operation.

---

# Frontend Technologies

## React Native

### Purpose

Used for developing the Teacher-Assisted Learning Portal for tablets.

### Features Utilized

- Component-Based UI Development
- State Management
- Navigation
- Responsive Interfaces
- Cross-Platform Development

### Provider

Meta Platforms, Inc.

---

## Bootstrap

### Purpose

Used for developing responsive interfaces for the Parent Monitoring Portal and Teacher Management Portal.

### Features Utilized

- Responsive Layout System
- UI Components
- Forms
- Navigation Menus
- Dashboard Layouts

### Provider

Bootstrap Team

# Educational and Accessibility Features

The following concepts are incorporated into the system but are not developed as standalone technologies by the researchers.

## Adaptive Learning

### Purpose

Adjust activity difficulty according to learner performance.

### Inputs

- Response Time
- Completion Rate
- Accuracy
- Previous Performance

---

## Positive Reinforcement System

### Purpose

Encourage learner engagement and participation.

### Features

- Reward Stickers
- Achievement Indicators
- Visual Rewards
- Encouraging Messages
