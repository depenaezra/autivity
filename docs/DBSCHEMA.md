# Database Schema

The **AutiVity** database is designed to support role-based access, adaptive learning, learner progress monitoring, and communication between the React Native learner application and the teacher web portal through a shared MySQL database.

## Database Overview

- **Database Name:** `autivity_db`
- **Database Management System:** MySQL 8.x
- **Architecture:** Relational Database
- **Normalization:** Third Normal Form (3NF)

---

## Tables

### 1. `users`

Stores authentication and account information for all system users.

| Column | Type | Constraints |
| :--- | :--- | :--- |
| user_id | BIGINT | PK, AUTO_INCREMENT |
| first_name | VARCHAR(100) | NOT NULL |
| last_name | VARCHAR(100) | NOT NULL |
| email | VARCHAR(150) | UNIQUE |
| password | VARCHAR(255) | NOT NULL |
| role | ENUM('admin','teacher','parent','learner') | NOT NULL |
| status | ENUM('active','inactive') | DEFAULT 'active' |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP |
